## Stack
- Flask 2.x backend (Python)
- Pandas + OpenPyXL para procesamiento Excel
- Frontend HTML/JS vanilla (sin framework)
- Puerto local: 5000

## Comandos
- `python app.py` — servidor de desarrollo
- `pip install -r requirements.txt` — instalar dependencias

## Convenciones
- Comentarios en español
- Rutas de API en /upload, /preview, /confirm, /merge
